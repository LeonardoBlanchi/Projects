# Market Making & Risk Control in MSFT Options (Python)

Group project on equity-options market making and market risk control.

## What we did
- Built a delta-hedged options portfolio on Microsoft (MSFT) using listed + OTC options
- Computed Greeks and tracked risk/P&L drivers (gamma, vega)
- Estimated 5-day VaR (95%) and ES (97.5%) with Monte Carlo
- Ran historical + hypothetical stress tests and monitored limits

- Report/slide deck
- Python notebooks/scripts for simulation and risk metrics
- Outputs (tables/figures)
